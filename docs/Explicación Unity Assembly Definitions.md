Resumidamente, en Unity existe un tipo de archivo llamado '_Assembly Definitoon Asset_'.
Este archivo una vez colocado dentro de una carpeta, **automáticamente** todos los scripts ubicados ahí, formaran parte de un Assembly.
Para poder interactuar con scripts de otros assemblies, es necesario añadirlo como referencia al Assembly que necesita esos scripts.
